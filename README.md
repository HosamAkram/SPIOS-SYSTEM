# Smart Pharmacy Inventory Optimization System (SPIOS)

## Domain
Pharmacy Inventory Management

## Role
Business Analyst

## Project Overview
SPIOS is a smart inventory management system designed to help pharmacies optimize stock levels, reduce expired medications, and improve reorder decisions using historical sales data and inventory analytics.

## Business Problems
- Overstocking of slow-moving products.
- Stock shortages of fast-moving medications.
- Financial losses due to expired products.
- Manual reorder decision-making.

## Key Features
- Product Management
- Batch Management
- Purchase Invoice Management
- Near Expiry Alerts
- Daily Replenishment Report
- Reorder Recommendation Engine
- Approval Workflow
- Inventory Dashboard
- Sales Dashboard
- Audit Logging
